## ⌨️ Setup
To run this project, install it locally using npm:
#### Cloning
```
git clone https://github.com/sfjgnjhdskgas/nike.git
cd nike
```
#### Installation
Install the project dependencies using npm:
```
npm install
```
#### Running the Project
```
npm run dev
```
